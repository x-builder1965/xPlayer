# xPlayer -動画プレイヤー- (xPlayer-windows)

## 概要
xPlayer は **ローカル動画の再生・編集** を中心に設計された、軽量でシンプルな Electron ベースの動画プレイヤーです。  
幅広い動画形式に対応し、プレイリスト管理、再生速度変更、カット編集、動画結合、ズーム機能などを備えています。  
ネット動画（YouTube / Twitch など）は外部ブラウザでポップアップ再生します。

## 主な機能
- **ローカル動画再生**  
  mp4, mkv, webm, avi, flv, mov, wmv, mpg, mpeg, ts, mts, m2ts, vob, ogv, 3gp, m4v, asf など広範囲の形式に対応  
  HTML5でネイティブ再生可能な形式はそのまま再生  
  非対応形式は **FFmpeg** を使って自動で mp4 に変換（日本語音声・字幕を優先）

- **プレイリスト管理**  
  プレイリスト（.amppl形式）の作成・編集・保存・読み込み  
  フォルダ丸ごと読み込み（再帰対応）  
  ドラッグ＆ドロップでファイル／フォルダ追加  
  並び替え（パス順・作成日時順・ランダム）  
  上へ／下へ移動、削除、クリア

- **再生制御**  
  再生／一時停止／停止／30秒送り戻し／前後動画移動  
  再生速度変更（0.25×〜2.0×）  
  ランダム再生（🔀）／繰り返し再生（全動画・1動画・なし）  
  設定はアプリ再起動後も記憶

- **カット編集 ✂️**  
  複数範囲指定で不要部分を削除 → 残りを結合して保存  
  IN/OUTマーク設定、タイムライン表示、フレーム単位微調整  
  高速モード（ストリームコピー）／高精細モード（再エンコード）を自動判別  
  保存形式：mp4（ダイアログで変更可）

- **動画結合 🎞️**  
  プレイリスト内の複数動画を結合  
  FPS自動検出＆多数決採用（Ver 3.54 以降）  
  統一フォーマット変換後、高速結合

- **ズーム＆移動 🔍**  
  -100%〜+500% のデジタルズーム  
  マウスドラッグでパン（画面移動）  
  ホイール／Ctrl+↑↓で調整、Ctrl+0でリセット  
  状態を記憶（動画切り替え・再起動時も維持）

- **スナップショット 📷**  
  現在のフレームを Windows Snipping Tool で撮影

- **ネット動画再生 🌐**  
  YouTube / Twitch / その他URLを外部ブラウザ（Chrome推奨）でポップアウト再生  
  クリップボードのURLを自動貼り付け

- **その他**  
  豊富なショートカットキー（ショートカットキーの詳細はヘルプ ❔で確認可能）
  コントロール自動非表示（3秒）  
  ドラッグ＆ドロップ対応  
  変換・編集時の一時ファイル自動クリーンアップ

## 動作環境
- Windows 10 / 11（推奨）
- 配布版は Electron パッケージング済み（Node.js 不要）
- 開発時は Node.js 18+ / npm が必要

## インストール・起動
1. 最新リリースから `xPlayer Setup x64.exe` をダウンロード
2. インストール後、デスクトップ／スタートメニューから起動
   - 動画ファイル（.mp4など）やプレイリスト（.amppl）をダブルクリックで関連付け再生も可能

### 開発・ビルド（ソースから）
```bash
git clone <repository-url>
cd xPlayer-windows
npm install
npm start          # 開発実行
npm run build      # Windowsインストーラー生成（dist-win に出力）
